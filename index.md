## GitHub Pages notes

After uploading these files to a GitHub repository:

1. Go to **Settings**
2. Open **Pages**
3. Under **Build and deployment**, choose **Deploy from a branch**
4. Select the **main** branch and the **/(root)** folder
5. Save

Your site will be published automatically.

## Suggested improvement before publishing

Add your screenshots into an `assets/` folder and link them from each page using Markdown images, for example:
